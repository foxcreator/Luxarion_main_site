# LUXARION — Брендбук

## 1. Опис студії

**Назва:** Luxarion

**Тип:** Game Development Studio

Luxarion — незалежна студія розробки відеоігор, що створює високоякісні інтерактивні продукти для глобальної аудиторії.

Студія працює з різними жанрами та платформами, поєднуючи креативний підхід до дизайну ігор із сучасними технологіями розробки.

Мета Luxarion — створювати оригінальні IP, впізнавані світи та геймплей, до якого гравці захочуть повертатися.

---

# 2. Сенс назви

**Lux** — світло, відкриття, сяйво

**Arion** — міфологічний символ сили та легенди

Разом:

**Luxarion — це світло, що створює нові світи та ідеї.**

---

# 3. Позиціонування бренду

Luxarion позиціонується як **преміальна незалежна студія розробки ігор**, що поєднує креативність, технології та сильну дизайн-філософію.

Основні фокуси студії:

• створення оригінальних IP
• розробка ігор у різних жанрах
• створення запам'ятовуваного геймплею
• преміальна якість продукту

Luxarion не обмежується одним жанром — студія експериментує з різними ігровими напрямками, зберігаючи власну творчу ідентичність.

### Основний логотип

Використовується у:
